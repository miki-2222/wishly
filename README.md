## サービス概要
購入予定のアイテムを管理し、計画的にアイテム購入ができるアプリです。

「発売日や予約日がバラバラで管理が大変！」「このグッズどこで予約してたっけ？」
そんな不安やめんどくさい管理を解消します。

購入したいアイテム、購入済みアイテムをまとめて管理するサービス。


## このサービスへの思い・作りたい理由
きっかけは本の購入忘れとBlu-rayの重複購入でした。
私はグッズ収集が好きなのですが、発売日や予約日がバラバラでしたり、予約期間が長いもの、そもそも発売日が大まかにしか決まっていないものと多種多様あり管理が大変でした。
しかも様々な購入サイトや店舗で発売される為、どの商品がどこから発売されるのかという点も混乱を極めます。
推し活ばかりに時間を取られるわけにはいかない。そんな中。重複購入と購入忘れをしてしまったのです。

欲しいアイテムなど絶対買い逃さないように＆複数購入しないようにしたい。種類関係なくまとめて管理したいという思いから作ろうと思いました。


## ユーザー層について
・様々なグッズなどを収集する人
・予約商品をよく買う人
・自分の好きなタイミングで購入する人
・所持品を種類関係なくまとめて管理したい人


## なぜこの層にしたか
・発売日や購入先がバラバラで、欲しいアイテムが沢山あるがゆえに購入したかどうか忘れてしまうことを防止することに役立ててほしいと考えたため。
・急な出費に備え、購入をギリギリにする方もおられるので、購入予定日を決めることが役に立つと考えたため。
・購入前後を一貫して管理できるため、様々なアプリを併用しなくても済むようにできるため。


## サービスの利用イメージ
【欲しい商品を見つけた！】
1. 商品情報を入力（商品名、価格、発売日など）
2. 予約又は購入したら、ボタンを押すだけ！
➞ 購入予定が一覧で確認でき、「これ予約したっけ？」という不安がなくなります

【持っている商品を確認したい】
1. 一覧から確認
2. 多い時は絞込
➞ 既に購入したか分からなくなった時、該当商品を一覧で確認できるため、重複購入を防げます

## サービスの差別化ポイント・推しポイント
#### **購入前後の一貫管理**
既存サービスは「所持品管理」か「欲しい物リスト」のどちらか一方が多いです。
- 本サービス: 購入前→購入後まで一貫して同じアプリで管理
- 実際の体験: 一覧画面でまとめて確認できるので購入漏れや重複購入がなくなる
#### **グッズ特有の「予約期間」に対応**
- 一般的なタスク管理アプリでは管理しきれない「予約開始日〜予約終了日」を管理
- 実際の体験: 「予約期間が3日間だけ」のような限定商品も買い逃す心配を減らす

#### **推し活以外にも使える汎用性**
- 本、ゲーム、家電など、ジャンル問わず「発売前商品」を管理
- 実際の体験: 「新刊の発売日いつだっけ？」「このゲーム予約してたっけ？」も解決

**競合比較表**
| サービス | 特徴 | 差別化ポイント |
| ---- | ---- | ---- |
| Mono-モノ管理 | 所持しているアイテムを確認できる。 |  予約含め、購入前後を一貫管理でき、重複購入を防げる安心感が得られる。 |
| 推しノート | 推しのグッズの発売日や支払い日が登録できる。 | 本、ゲーム、家電など、ジャンル問わず管理できる。「新刊の発売日いつだっけ？」「このゲーム予約してたっけ？」も解決。 |
